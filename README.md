# ⚡ Smart EV Charge

Smart EV Charge is a full-stack web-based Electric Vehicle Charging Management System designed to simplify EV charging operations for EV owners, charging center owners, and system administrators.

## 🚀 Features

### 👤 EV Owner Module
- User Registration & Login
- Find Nearby Charging Stations
- GPS-Based Location Tracking
- Book Charging Slots
- View Booking History
- Payment Management
- Charging Activity Tracking
- Ratings & Reviews

### 🏢 Charging Center Owner Module
- Center Registration & Management
- Approve/Reject Bookings
- Manage Charging Slots
- Monitor Charging Sessions
- Revenue Tracking
- Reports & Analytics

### 👨‍💼 Administrator Module
- Manage Users
- Manage Charging Centers
- Monitor Bookings
- Payment Management
- System Reports
- Dashboard Analytics

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/isuru6106/Smart-EV-Charge.git
```

2. Copy project to XAMPP htdocs

```text
C:\xampp\htdocs\
```

3. Start Apache and MySQL from XAMPP

4. Create Database

```sql
CREATE DATABASE ev_charging_system;
```

5. Import

```text
database/ev_charging_system.sql
```

6. Open in browser

```text
http://localhost/ev_charging/
```
